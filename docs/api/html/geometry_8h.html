Computes the control points of the panels where the no penetration boundary condition is satisfied at every time instant.

This function determines the midpoints of each panel, which serve as control points. It also needs to be updated at every time instant. To calculate control points

Parameters

Generates the initial nodal coordinates using cosine clustering.

This function discretizes the chord in a nonlinear manner to improve panel distribution. near the leading and trailing edges. The term initial refers to the configuration of airfoil at time t=0 in inertial frame of reference which is also the origin of the Therefore this function needs to be updated only once in the main function that too outside the time loop.

Parameters
x0Reference to a VectorXd to store the x-coordinates of the nodes.
y0Reference to a VectorXd to store the y-coordinates of the nodes.

Computes the instantaneous nodal coordinates in the inertial frame.

This function transforms the initially generated nodal coordinates to the inertial frame based on the angle of attack and time. This needs to be called at every time instant to update the nodal coordinates of the panels on airfoil surface.

Parameters
x0Reference to the x-coordinates of the nodes in the body-fixed frame.
y0Reference to the y-coordinates of the nodes in the body-fixed frame.
x_ppReference to a VectorXd to store the transformed x-coordinates.(Inertial frame)
y_ppReference to a VectorXd to store the transformed y-coordinates.(Inertial frame)
alphaAngle of attack in radians.
tTime instant.
